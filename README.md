# Preparations
1. Install chrome [driver](https://chromedriver.chromium.org/home) and place it in the project root
2. Copy `config.ini.dist` into `config.ini`
3. Place your settings inside `config.ini`
4. Run `python3.exe -m venv venv`
5. Activate venv by running `venv\bin\activate.exe`
6. run `pip3.exe install -r requirements.txt`
7. run `python.exe main.py`
